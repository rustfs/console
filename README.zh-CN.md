## 特性

- 基于Vue3、Vite5、TypeScript、NaiveUI、Unocss等最新技术栈开发
- 基于[alova](https://alova.js.org/)封装和配置，提供统一的响应处理和多场景能力
- 黑暗主题适配, 界面样式保持Naive风格
- 仅在提交时进行eslint校验，没有过多限制，开发更简便
- 界面样式布局灵活可配置
- 多语言（i18n）支持

## 安装使用

本地开发环境建议使用 pnpm 8.x 、Node.js 必须 20.x

```bash
# install dependencies
pnpm i

# Run
pnpm dev

# Build product
pnpm build

```
